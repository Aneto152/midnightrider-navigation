# Plugins Documentation

Complete reference for all 9 Signal K plugins on MidnightRider J/30.

## The 9 Plugins

### Core Navigation (2)
- **WIT IMU USB v2.1.0** - 9-axis attitude sensor via USB serial
- **WIT IMU Bluetooth v2.0.0** - Same sensor, Bluetooth LE interface

### Analysis (4)
- **Wave Height v2.1.0** - Ocean wave analysis from acceleration (3D magnitude)
- **Performance Polars** - J/30 efficiency vs polar diagram
- **Sails Management v2.0** - Intelligent sail recommendations
- **Astronomical** - Sun/Moon/Tide calculations

### Support (3)
- **Loch Calibration** - Speed through water accuracy
- **Current Calculator** - Drift & tidal current estimation
- **CPU Temperature** - System health monitoring
