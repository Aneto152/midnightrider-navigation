## 🎯 VOILES STANDARDS J/30

### Inventaire Typique

**Mainsail:**
- One mainsail with 1-2 reefing points
- Reefing system: mechanical or line-based
- Boom-mounted outhaul & vang
- Typical reefing reduces area by ~30-40% per reef point

**Jib Setup:**
- Primary: Working Jib (196 sq ft) — main workhorse
- Optional: Genoa (294 sq ft) — light air upgrade
- No storm jib typical (J/30 can handle gale with reefed main + working jib)

**Spinnaker:**
- One symmetric spinnaker (704 sq ft)
- Pole-launched (if crew comfortable)
- Often used in club racing, occasionally in cruising
- Risky single-handed (not recommended without experience)

## ⛵ RÉGLAGES TYPIQUES J/30 PAR CONDITION

### Based on: Sailing experience + class notes + general 30ft cruiser-racer practices

#### LIGHT AIR (< 7kt)

```
Goal: Maximum speed, good heel (12-15°)
Sail Config: Full main + Genoa (if available, else full jib)
Mainsheet: Ease to match jib
Outhaul: Loose (draft in the sail)
Vang: Light tension only (prevent boom drop)
Heel Target: 12-14°
Notes:
  • Genoa adds ~2-3 sq ft more area than working jib
  • Every knot counts — trim obsessively
  • May need body weight for heel
  • Spinnaker not viable (unstable)

Heel Issues:
  < 10°: Under-powered, need more sail or trim for power
  10-14°: Ideal, maximum VMG
  > 14°: Over-powered (rare in light air)
```

**Light Air Checkpoints:**
- [ ] Check genoa available and ready?
- [ ] Mainsheet eased smoothly (no chafe)?
- [ ] Outhaul loose enough (hollow leech)?
- [ ] Vang minimal tension?
- [ ] Crew weight forward for light displacement?

---

#### MEDIUM WIND (7-12kt) — "IDEAL CONDITIONS"

```
Goal: Balanced speed & control, good VMG
Sail Config: Full main + Working jib
Mainsheet: Tight enough for good leech, not over-sheeted
Outhaul: Medium tension (flatter draft)
Vang: Medium (control boom twist)
Heel Target: 16-18° (optimal)
Notes:
  • This is the J/30's sweet spot (why it's fun to race!)
  • Focus on smooth trim, not extreme angles
  • Performance is rewarding — 6-7 knots typical
  • Smooth conditions = full speed

Heel Issues:
  14-16°: Very good, maximize VMG
  16-18°: Ideal, stable and fast
  18-20°: Getting high, consider reef soon
  > 20°: Needs attention, reef or adjust trim
```

**Medium Wind Checkpoints:**
- [ ] Working jib up (genoa should be down)?
- [ ] Mainsheet tension for clean leech?
- [ ] Outhaul creating flatter draft?
- [ ] Vang preventing excessive boom twist?
- [ ] Helm balanced (slight weather helm OK)?

---

#### FRESH WIND (12-16kt) — "TIME TO REEF DECISION"

```
Goal: Control heel, maintain stability
Sail Config: Full main + Working jib (monitor for reef)
Mainsheet: Tight, control power
Outhaul: Tight (flatter, depowered)
Vang: Firm (reduce boom lift, control twist)
Heel Target: 18°
Notes:
  • Heel starting to approach limit (20-22° is max safe)
  • Consider 1-reef if:
    - Heel consistently > 18°
    - Gusts pushing to 20°+
    - Wave height > 1.0m
  • Reefed main keeps boat controllable
  • Working jib stays (no change needed)

Heel Issues:
  16-18°: Good, manageable
  18-20°: Getting high, monitor gusts
  > 20°: Should have reefed already!

When to Reef Main (1 point):
  • Sustained TWS > 14kt AND heel > 18°
  • OR gusts reaching 18-20kt
  • OR seas building > 1.0m height
  • Rule: "If you're thinking about it, do it" — safer
```

**Fresh Wind Checkpoints:**
- [ ] Monitor heel trend (rising = reef soon)
- [ ] Tighten outhaul progressively
- [ ] Watch for consistent gusts > 16kt
- [ ] Have reef points visible and clear?
- [ ] Crew ready to execute reef?

---

#### STRONG WIND (16-20kt) — "REEF MAIN, REDUCE POWER"

```
Goal: Control heel < 20°, safety first
Sail Config: 1-Reef main + Working jib
Mainsheet: Tight, trim for power management
Outhaul: Tight (depowered)
Vang: Very firm (reduce power)
Heel Target: 18-20° (conservative)
Notes:
  • Should have 1 reef already if sustained > 14kt
  • J/30 handles 16-20kt well with 1 reef + working jib
  • Good stability, predictable helm
  • Second reef needed only in extreme conditions
  • Spinnaker: ABSOLUTELY NOT (douse if still up)

Heel Issues:
  18-20°: Good, stable, safe
  > 20°: Already past where you should be
  > 22°: CRITICAL, need to reduce immediately

When to Consider 2nd Reef:
  • TWS sustained > 18kt
  • OR gusts > 22kt
  • OR seas > 1.5m with rough motion
  • OR helm very heavy (weather helm > 25°)
```

**Strong Wind Checkpoints:**
- [ ] 1st reef confirmed down?
- [ ] Working jib set (no bigger jib)?
- [ ] Outhaul & vang maximally tight?
- [ ] Heel staying < 20°?
- [ ] Crew comfortable? (physiology check)

---

#### GALE (> 20kt) — "SURVIVAL MODE"

```
Goal: Heel < 22° ABSOLUTE MAXIMUM
Sail Config: 2-Reef main + Working jib (or drop to storm setup)
Mainsheet: Extreme tension, minimal power
Outhaul: Fully tight (maximum depower)
Vang: Maximum (reduce any excess power)
Heel Target: 20-22° (safety margin only)
Notes:
  • J/30 can handle 20-25kt with 2 reefs + working jib
  • NOT recommended for casual crew
  • Requires strong helmsman + good communication
  • Consider dropping mainsail entirely if:
    - Heel exceeds 22°
    - Gusts > 25kt
    - Seas > 2.0m
  • Many J/30s furl jib & run under 2-reef main alone

Heel Issues:
  20-22°: At limit, monitor every moment
  > 22°: DROP SAIL IMMEDIATELY (emergency)
  > 25°: Already lost control, reduce everything

When to Drop Main / Run Under Jib Alone:
  • Heel > 22° despite 2 reefs
  • Gusts > 25kt (unpredictable)
  • Seas > 2.0m (stability concerns)
  • Crew exhausted (safety first)

Gale Configuration Options:
  Option A: 2 Reefs + Working Jib (if confident, experienced)
  Option B: Furl Jib + 2 Reefs main (easier handling, more power)
  Option C: Under Jib Alone (safest, slowest)
```

**Gale Checkpoints:**
- [ ] Both reefs confirmed down?
- [ ] Jib furled or minimal (working jib only)?
- [ ] Heel NEVER exceeds 22°?
- [ ] Crew alert and rested?
- [ ] Exit strategy clear (harbor, protected water)?

## 📊 J/30-SPECIFIC MATRIX (Based on Specs)

### BEATING (Au Près)

```
             LIGHT   LIGHT_AIR  MEDIUM   FRESH   STRONG   GALE
             <4kt    4-6kt      7-11kt   12-15   16-19    >=20
MAIN         FULL    FULL       FULL     FULL    1-REEF   2-REEF
JIB          GENOA   FULL       WORKING  WORKING WORKING  WORKING
HEEL TARGET  12°     14°        16°      18°     20°      22°
NOTES        Rare    Genoa ok   Ideal    Good    Monitor  Gale
```

**J/30 Specifics:**
- Genoa (294 sq ft) vs Working Jib (196 sq ft) = +98 sq ft gain
- Small gain in light air (2-3% speed), worth the effort
- 1-Reef reduces main from 247 to ~165-175 sq ft
- Heel management is KEY (pitch sensitivity in this range)

---

### REACHING (Largue)

```
             LIGHT   LIGHT_AIR  MEDIUM   FRESH   STRONG   GALE
             <4kt    4-6kt      7-11kt   12-15   16-19    >=20
MAIN         FULL    FULL       FULL     FULL    1-REEF   2-REEF
JIB          GENOA   GENOA      WORKING  WORKING WORKING  WORKING
SPINNAKER    READY   UP         READY    DOWN    DOWN     DOWN
HEEL TARGET  15°     16°        18°      20°     22°      20°
NOTES        Power   Power      Optimal  Spicy   Risky    Bare
```

**J/30 Specifics:**
- Spinnaker (704 sq ft!) is HUGE compared to main (247 sq ft)
- Only experienced crews should hoist in > 12kt wind
- Reaching is power boat-like (heel can build fast)
- 2° more heel than beating because reaching is forgiving

---

### RUNNING (Vent Arrière)

```
             LIGHT   LIGHT_AIR  MEDIUM   FRESH   STRONG   GALE
             <4kt    4-6kt      7-11kt   12-15   16-19    >=20
MAIN         FULL    FULL       FULL     FULL    1-REEF   DOWN
JIB          OUT     OUT        OUT      OUT     POLED    POLED
SPINNAKER    UP      UP         READY    DOWN    DOWN     DOWN
HEEL TARGET  12°     13°        15°      16°     18°      14°
NOTES        Stable  Stable     Optimal  Tricky  Danger   Bare
```

**J/30 Specifics:**
- Jib out = wing-on-wing (stabilization)
- Running is HIGHER RISK than reaching (jibe risk)
- Spinnaker is incredible (704 sq ft!), but needs expertise
- Heel target LOWER on run (stability vs power trade-off)
- Many cruisers avoid running under spinnaker (safer with jib out)

## 🧮 HEEL ANGLE TARGETS (J/30-Specific)

### Optimal Heel by Condition

| Wind Class | Upwind | Reaching | Downwind | Why |
|------------|--------|----------|----------|-----|
| LIGHT (< 7kt) | 12-14° | 15-16° | 12-13° | Max speed, stability in light |
| MEDIUM (7-12kt) | 16-18° | 18° | 15° | VMG optimal, balanced |
| FRESH (12-16kt) | 18° | 20° | 16-18° | Heel approaching limit |
| STRONG (16-20kt) | 18-20° | 20-22° | 18° | Safety priority, helm control |
| GALE (> 20kt) | 20-22° | N/A | 14-16° | SAFETY ONLY, minimize power |

### Why These Angles?

**Upwind (Beating):**
- < 12°: Under-powered, missing speed
- 12-18°: Sweet spot (VMG optimal)
- > 20°: Excessive, performance drops, safety risk
- J/30 starts to lose efficiency > 20° heel

**Reaching:**
- < 15°: Under-powered on reach
- 15-20°: Optimal (maximum speed)
- > 22°: Risk of broaching (J/30 susceptible in rough seas)

**Downwind:**
- < 12°: Under-powered, rolling
- 12-18°: Good stability
- > 20°: Risk of unintended jibe, balance loss

## ⚖️ REEFING SPECIFICS (J/30)

### Reef Points (Typical J/30 Setup)

| Reef | Mainsail Area Remaining | Boom Position | Usage |
|------|------------------------|----------------|-------|
| Full | 247 sq ft (100%) | Normal | < 14kt sustained |
| 1st Reef | ~165-175 sq ft (70%) | ~2-3 ft higher | 14-18kt sustained |
| 2nd Reef | ~100-120 sq ft (45%) | ~4-5 ft higher | 18-25kt sustained |

### Reefing Procedure (Mechanical)

**Typical J/30 has boom-mounted reefing system:**

```
1st Reef (from Full to 1-Reef):
  1. Release outhaul (let boom forward)
  2. Ease mainsheet
  3. Tighten reef downhaul (cringle to boom)
  4. Re-tension outhaul
  5. Retrim mainsheet
  Time: 2-3 minutes (practiced crew)

2nd Reef (from 1-Reef to 2-Reef):
  1. Release outhaul again
  2. Ease mainsheet
  3. Tighten 2nd reef downhaul
  4. Re-tension outhaul
  5. Retrim mainsheet
  Time: 2-3 minutes
  
Total time (Full to 2-Reef): 4-6 minutes
```

## 🎯 CREW COMFORT & LIMITS

### Experienced Crew (10+ ocean racing hours)

- Can handle full main + working jib in 16-18kt
- Comfortable with 1-reef in 18-20kt
- Can manage spinnaker hoist/douse in 10-14kt
- Can execute 2-reef in 20-25kt with communication
- Knows when heel is getting excessive

### Cruising Crew (occasional sailing)

- Comfortable with full sail up to 14kt
- Ready to reef at first sign (14-15kt)
- Prefer conservative (reef earlier = less stress)
- Spinnaker = optional (many avoid)
- Safety over performance = good philosophy

### Single-Handler Limits

- Full sail in 12kt max (too much to manage alone)
- Reef early (14kt trigger)
- No spinnaker (too dangerous solo)
- Heavy weather = anchoring/running to shelter
- Safety = paramount

## 🚨 J/30 SAFETY LIMITS

### Hard Limits (Never Exceed)

| Parameter | Limit | Consequences |
|-----------|-------|--------------|
| Heel Angle | 22-25° | Beyond this = capsize risk for J/30 |
| Wave Height | 2.5m+ | Design limit exceeded, risk of dismasting |
| Wind Gust | 28kt+ | Unpredictable forces, high risk |
| Duration in Gale | 4+ hours | Crew fatigue exceeds safety margin |

### Warning Signs (Reduce Sail Immediately)

- [ ] Helm very heavy (hard to steer)
- [ ] Excessive leeway (sideways slipping)
- [ ] Boat not righting naturally (heel is too much)
- [ ] Water coming over deck (freeboard exceeded)
- [ ] Crew expressing fear or discomfort
- [ ] Mast movements look wrong (compression risk)

## 📋 J/30 REEFING DECISION TREE

```
Wind Speed Rising...

Is TWS < 14kt sustained?
  → Keep full sail, monitor
  
Is TWS 14-16kt sustained?
  → REEF 1
  
Is TWS 16-18kt sustained?
  → REEF 2 (or check if 1-reef is enough)
  
Is TWS > 18kt sustained?
  → REEF 2 minimum, consider dropping main

Are gusts exceeding average wind by > 5kt?
  → Reef one point earlier (add safety margin)

Are seas building (waves > 1.5m)?
  → Reef one point earlier (motion = fatigue)

Is crew inexperienced?
  → Reef one point earlier (safety first)

Is crew exhausted?
  → Reef or harbor up immediately (fatigue kills)
```
