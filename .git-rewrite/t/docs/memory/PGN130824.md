# PGN 130824 — B&G Performance Data

## Overview

- **PGN:** 130824 (0x1FF08) — propriétaire B&G
- **Type:** Performance data stream
- **Direction:** UNIDIRECTIONNEL (Signal K → Vulcan, jamais l'inverse)
- **Transport:** NMEA2000 via YDNU-02
- **Destination:** Afficheur B&G Vulcan
- **Plugin:** `signalk-bandg-performance-plugin`

## Architecture

```
Signal K Server
    ↓
signalk-bandg-performance-plugin
    ↓ (génère PGN 130824)
NMEA2000 Bus (YDNU-02)
    ↓
Afficheur B&G Vulcan
```

## Signal K Sources

| Path | Data | Unit | Notes |
|------|------|------|-------|
| `performance.targetSpeed` | Polar target speed | knots | Vitesse cible selon polaire |
| `performance.targetVMG` | VMG cible | knots | VMG optimal pour angle courant |
| `performance.velocityMadeGoodRatio` | % polaire | % | VMG actuel / VMG cible × 100 |
| `performance.beatAngle` | Angle optimal au près | ° | Meilleur angle au près (polaire) |
| `navigation.racing.layline` | Layline tribord | ° | Angle vers marque suivante |

## What It Does

1. **Signal K calcule** les données de perf en temps réel:
   - Analyse polaire ORC
   - Calcul VMG vs angle au vent
   - Layline vers marques

2. **Plugin B&G** formatte tout dans PGN 130824

3. **YDNU-02** transmet sur NMEA2000

4. **Vulcan** affiche les données en bateau

## Key Points

- ✅ **Unidirectionnel:** Signal K → Vulcan seulement
- ✅ **Real-time:** Mise à jour en continu
- ✅ **Proprietaire:** Format B&G (pas d'autre afficheur compatible)
- ❌ **Pas de retour:** Vulcan n'envoie rien à Signal K
- ⚠️ **Dépend de:** Calculs Signal K performants + polaire ORC chargée
